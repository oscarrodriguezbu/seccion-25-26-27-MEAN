
# Backend en Nest

1. Tener en cuenta que se necesita:
Docker Desktop corriendo con la imagen de mongo:5.0.16 y Mongo Compass conectado a localhost:27017

2. Copiar el ```.env.template``` y renombrarlo a ```.env```

3. Comandos:

```
## Correr docker:
docker compose up -d

## Correr proyecto:
npm run start:dev

```
